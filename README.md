# Rechat

## Setup
* Clone the repository `git clone https://github.com/Team3Webchat/rechat.git`
* Make sure all dependencies are installed
```sh
  npm install
  cd server
  npm install
  cd ../ui
  npm install
```
* Setup database todo
* Seed database todo

## Running 
### Server
Be sure to be in the server directory
* npm run start

### UI
Be sure to be in the ui directory
* npm run start
